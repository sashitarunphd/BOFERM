Bio-Inspired Optimization Framework for Energy-Aware Resource Management in Cloud–Edge Systems

This repository contains the dataset used for evaluating BOFERM, a multi-stage hybrid bio-inspired optimization framework that integrates ACO, PSO, ABC, and FA for energy-efficient resource management in heterogeneous cloud–edge systems.

The dataset enables reproducible experiments for energy consumption, SLA violations, migration overhead, and workload behavior across cloud–edge nodes.
